Q1 - Problem: You are given a string (STR) of length N, consisting of only the lower case English alphabet. Your task is to remove all the duplicate occurrences of characters in the string.
Input : abcadeecfb
Output : abcdef

ANSWER :

        let strings="abcadeecfb";
        const stringop = new Set();
        for(let i=0;i < strings.length;i++){
              stringop.add(strings[i]);
        }
        console.log(...stringop);

Q2 - Problem: You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps)
Input: abcadeecfb
Output:
a=2
b=2
c=2
d=1
e=2
f=1

 ANSWER :
        
        let inputstrings="abcadeecfb";
        let outputstrings = new Map();
        for(let i=0;i < inputstrings.length;i++){
            if(outputstrings.has(inputstrings[i])){
                outputstrings.set(inputstrings[i],Number(outputstrings.get(inputstrings[i]))+1);
            }
            else{
                outputstrings.set(inputstrings[i],1);
            }
        }
        for(const [outputkey,outputvalue]of outputstrings){
            console.log(outputkey +" = " + outputvalue);
        }
